# ShahMaghz — آجیل و خشکبار ممتاز ایرانی

فروشگاه آنلاین لوکس آجیل، خشکبار و جعبه‌های هدیه، با تجربه‌ای شاهانه.

## شروع سریع

```bash
# نصب وابستگی‌ها
npm install

# تنظیم متغیرهای محیط
cp .env.example .env
# مقادیر DATABASE_URL و سایر کلیدها را وارد کنید

# اجرای Prisma
npx prisma generate
npx prisma db push

# اجرای پروژه
npm run dev
```

## تکنولوژی‌ها

- **فرانت‌اند:** Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **پایگاه داده:** PostgreSQL + Prisma ORM
- **احراز هویت:** NextAuth.js (موبایل + OTP)
- **پرداخت:** زرین‌پال (IRR)
- **کش:** Redis
- **ذخیره‌سازی رسانه:** S3-compatible

## فازهای توسعه

- **فاز ۱ — پایه:** ✅ ساختار پروژه، اسکیمای دیتابیس، سیستم طراحی، صفحه اصلی، صفحه محصول
- **فاز ۲ — تجارت:** 🔄 سبد خرید، تسویه حساب، زرین‌پال، احراز هویت OTP
- **فاز ۳ — محتوا و وفاداری:** 🔄 مجله، باشگاه وفاداری، کدهای تخفیف
- **فاز ۴ — پنل مدیریت:** 🔄 مدیریت محصولات، سفارش‌ها، مشتریان
- **فاز ۵ — بهینه‌سازی:** 🔄 تست پرداخت، SEO، بهینه‌سازی کارایی

## زبان و RTL

تمام متن‌های رابط کاربری به زبان فارسی است. سایت به صورت RTL-native طراحی شده است:
- `dir="rtl"` در سطح سند
- استفاده از خصوصیات منطقی Tailwind (`ms-`, `me-`, `ps-`, `pe-`)
- ارقام فارسی برای قیمت‌ها و وزن‌ها
- فونت Vazirmatn برای متن فارسی
