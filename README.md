# ShahMaghz — آجیل و خشکبار ممتاز ایرانی

فروشگاه آنلاین لوکس آجیل، خشکبار و جعبه‌های هدیه، با تجربه‌ای شاهانه.

## شروع سریع

```bash
# نصب وابستگی‌ها
npm install

# اجرای پروژه در محیط توسعه
npm run dev

# ساخت پروژه برای تولید
npm run build

# اجرای پروژه ساخته‌شده
npm start
```

پروژه در `http://localhost:3000` اجرا می‌شود.

## استقرار در Vercel

### روش ۱: از طریق داشبورد Vercel

۱. به [vercel.com](https://vercel.com) بروید و وارد شوید.
۲. روی «New Project» کلیک کنید.
۳. ریپازیتوری `ShahMaghz` را از GitHub انتخاب کنید.
۴. تنظیمات به‌صورت خودکار شناسایی می‌شوند (Next.js).
۵. روی «Deploy» کلیک کنید.

### روش ۲: از طریق CLI

```bash
# نصب Vercel CLI
npm i -g vercel

# استقرار از پوشه پروژه
vercel

# برای استقرار نهایی (production)
vercel --prod
```

### متغیرهای محیط (اختیاری)

پروژه بدون نیاز به متغیرهای محیط کار می‌کند (داده‌ها در client-side ذخیره می‌شوند).
برای اتصال به پایگاه داده واقعی، متغیرهای زیر را در Vercel تنظیم کنید:

```
DATABASE_URL=postgresql://...
NEXTAUTH_SECRET=...
NEXTAUTH_URL=https://your-domain.vercel.app
```

## پنل مدیریت

- **آدرس:** `/admin`
- **نام کاربری:** `admin`
- **رمز عبور:** `shahmaghz1403`

> ⚠️ در محیط تولید، این اطلاعات را با احراز هویت واقعی جایگزین کنید.

## تکنولوژی‌ها

- **فرانت‌اند:** Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **پایگاه داده:** PostgreSQL + Prisma ORM (اسکیمای موجود، اتصال در فاز بعدی)
- **احراز هویت:** Zustand + localStorage (mock) — آماده برای ارتقا به NextAuth.js
- **پرداخت:** در حال حاضر غیرفعال — آماده برای اتصال زرین‌پال
- **استقرار:** Vercel

## زبان و RTL

تمام متن‌های رابط کاربری به زبان فارسی است. سایت به صورت RTL-native طراحی شده است:
- `dir="rtl"` در سطح سند
- استفاده از خصوصیات منطقی Tailwind (`ms-`, `me-`, `ps-`, `pe-`)
- ارقام فارسی برای قیمت‌ها و وزن‌ها
- فونت Vazirmatn برای متن فارسی
