## شروع سریع

```bash
# نصب وابستگی‌ها
npm install

# تنظیم متغیرهای محیط
cp .env.example .env.local
# مقادیر را از Neon و Cloudinary وارد کنید

# اجرای Prisma (جداول قبلاً ساخته شده‌اند)
npx prisma generate

# مشاهده پایگاه داده
npx prisma studio
# باز می‌شود در: http://localhost:5555

# اجرای پروژه
npm run dev
# باز می‌شود در: http://localhost:3000
```

## تکنولوژی‌ها

- **فرانت‌اند:** Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **پایگاه داده:** Neon (Serverless PostgreSQL) + Prisma ORM
- **تصاویر و CDN:** Cloudinary (بهینه‌سازی خودکار WebP/AVIF)
- **احراز هویت:** NextAuth.js (موبایل + OTP) — فاز ۲
- **پرداخت:** زرین‌پال (IRR) — فاز ۲
- **کش:** Redis (اختیاری)

## پایگاه داده (Neon PostgreSQL)

### جداول ایجاد شده (۱۳ عدد)

**احراز هویت و کاربران:**
- `users` — حساب‌های کاربری (مشتری و ادمین)
- `otp_codes` — کدهای تأیید موبایل
- `addresses` — آدرس‌های ارسال

**کاتالوگ محصولات:**
- `categories` — دسته‌بندی‌ها (سلسله‌مراتبی)
- `products` — محصولات اصلی
- `product_variants` — انواع (وزن‌ها، قیمت‌ها، موجودی)
- `reviews` — نظرات و امتیازها

**تجارت الکترونیک:**
- `orders` — سفارش‌های مشتریان
- `order_items` — اقلام سفارش
- `discount_codes` — کدهای تخفیف

**وفاداری:**
- `loyalty_accounts` — امتیازهای مشتریان
- `loyalty_transactions` — تاریخچه امتیاز

**محتوا:**
- `blog_posts` — مقالات مجله

### اتصال به پایگاه داده

```bash

npx prisma studio

# به‌روزرسانی اسکیما

npx prisma generate
```

## زبان و RTL

تمام متن‌های رابط کاربری به زبان فارسی است. سایت به صورت RTL-native طراحی شده است:
- `dir="rtl"` در سطح سند
- استفاده از خصوصیات منطقی Tailwind (`ms-`, `me-`, `ps-`, `pe-`)
- ارقام فارسی برای قیمت‌ها و وزن‌ها (`toPersianDigits`)
- فونت Vazirmatn برای متن فارسی
- فرمت‌دهی قیمت فارسی (`formatPrice`)

## هویت برند

### رنگ‌ها
- **سبز جنگلی** (Forest): رنگ اصلی برند (تیره، شیک)
- **طلایی** (Gold): رنگ تأکیدی (لوکس، ممتاز)
- **عاج** (Ivory): رنگ پس‌زمینه (گرم، طبیعی)
- **کرافت** (Kraft): رنگ بافت (ارگانیک، دست‌ساز)

### تایپوگرافی
- **نمایشی**: Vazirmatn Bold (عناوین)
- **متن**: Vazirmatn Regular (محتوا)
- **پشتیبانی کامل RTL**

### زبان طراحی
- **لوکس**: لهجه‌های طلایی، مهرهای موم، نشان‌های ممتاز
- **طبیعی**: بافت‌های کرافت، اشکال ارگانیک
- **فارسی**: چیدمان RTL، اعداد فارسی، محتوای فارسی
- **سردبیری**: چیدمان‌های نامتقارن، تصاویر بزرگ
